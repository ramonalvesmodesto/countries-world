# Países do mundo
Projeto que exibe todos os países, onde você pode filtrar por nome e região.

## Visão Geral :notebook:

### O desafio
Os usuários devem ser capazes de:

- Visualizar todos os países
- Filtrar por países
- Filtrar por região
- Visualizar informações de cada país
- Mudar a cor do site
